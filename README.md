# 植物激素教学网站

一个交互式的植物激素学习平台，包含动画演示、知识问答和实验模拟等功能。

## 功能特点

- 植物激素作用机制的动画演示
- 智能问答系统
- 知识测试
- 实验模拟
- 视频教学

## 部署说明

### 本地开发环境

1. 安装依赖：
```bash
npm install
```

2. 启动开发服务器：
```bash
npm start
```

### 生产环境部署

1. 构建项目：
```bash
npm run build
```

2. 部署方式：

- 静态托管：将 dist 目录下的文件部署到静态托管服务（如 GitHub Pages、Vercel、Netlify 等）
- 服务器部署：将 dist 目录下的文件上传到 Web 服务器
